# Przełącznik z animowanym emotikonem
### Rozmiar przełącznika jest w pełni uzależniony od niestandardowej właściwości:
```css
 --toggle-width
```
### Komponent ten posiada animacje, która przeprowadza transformacje wyglądu emotikona.
